Supplementary Table X. Diffusion MRI quality metrics (IQMs) derived from QSIPrep. Metrics summarize motion, signal quality, diffusion consistency, fiber orientation validity, and slice artifacts. Variable names correspond to QSIPrep outputs. Prefix notation [raw|t1|t1post]_ indicates the processing stage at which the metric was computed (raw diffusion data, preprocessed data prior to B1 bias correction, or preprocessed data after B1 bias correction).
